## 2025-02-28 - Visually hidden skip-to-content link bypassing sticky global header
**Learning:** For users relying on keyboard navigation, repetitive and numerous focusable elements in a sticky global header can be frustrating. Bypassing them allows users to jump directly to the primary page content.
**Action:** Implemented a 'skip to main content' link (with standard sr-only pattern via `skipToContentPolicy` in `app/constants/policy.ts`) as the first interactive element in `app/layout.tsx`. It targets a `<main id="main-content" tabIndex={-1} className="focus:outline-none">` element. This approach ensures it gets focus when interacted with, scrolling the user to it, but avoids displaying an ugly, unwanted focus ring as the main content isn't inherently an interactive component.
