## 2026-04-07 - Added Skip-to-Content Link
**Learning:** This app's navigation and layout was missing a 'skip to main content' link, which is critical for keyboard and screen reader accessibility so users can bypass the sticky header nav.
**Action:** Always ensure the RootLayout contains a visually hidden 'skip-to-content' link that jumps to the main application body, especially when sticky global headers are present.
