## 2024-05-18 - Internal Link Skip to Content Targets and Focus Management
**Learning:** Using `tabIndex={-1}` combined with `focus:outline-none` on an internal link target element (like `#main-content`) is critical. Without `tabIndex={-1}`, the element cannot programmatically receive focus from an internal link click. Without `focus:outline-none`, clicking a skip link leaves an unwanted visual focus ring around the entire target element (e.g., the whole `<main>` block), degrading visual polish while providing no real accessibility benefit since the structure is for layout only.
**Action:** Always add both `tabIndex={-1}` and `focus:outline-none` (or equivalent styles) to non-interactive container elements when they are used as targets for internal semantic jump links like 'skip to content'.
