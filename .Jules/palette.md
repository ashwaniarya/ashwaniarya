## 2025-02-27 - Skip to Main Content Link Pattern
**Learning:** In a Next.js app with a sticky global header, keyboard-only users and screen reader users need a visually hidden 'skip to main content' link. The pattern requires placing the link right after the opening `<body>` tag, making it focusable (`sr-only focus:not-sr-only`), and ensuring the target container (`<main>`) has an `id="main-content"`, `tabIndex={-1}`, and `focus:outline-none` so it can be programmatically focused without showing an unwanted focus ring.
**Action:** Use this established skip-link pattern (`skipToContentPolicy` in `policy.ts`) whenever creating a new top-level layout or when a page structure changes significantly, ensuring the main content area remains easily accessible for keyboard navigation.
