## 2024-04-27 - Skip to Content Internal Link Target A11y Focus
**Learning:** When creating "skip to content" links that target non-interactive containers (like a `<main>` element), the target container must have `tabIndex={-1}` applied so it can receive programmatic focus and allow sequential keyboard navigation to proceed from that point. Additionally, we must ensure `focus:outline-none` is added to prevent native focus rings from appearing around the entire block when it gets focused via the skip link.
**Action:** Always add `tabIndex={-1}` and a utility like `focus:outline-none` to anchor targets (`<main id="main-content">`, `<section id="some-section">`) used for programmatic keyboard skipping within this app to ensure smooth accessible navigation without visual glitches.
