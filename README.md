Personal website built with **Next.js App Router**, **TypeScript**, and **Tailwind CSS**.

## Getting started

- Install dependencies:

```bash
npm install
```

- Run the development server:

```bash
npm run dev
```

- Open `http://localhost:3000`
- Key routes:
  - `/` (home)
  - `/work/example-case-study` (example work detail page)
